Тестирование API

Получить все локации:

GET /api/locations/

Получить текущую локацию пользователя:

GET /api/user-locations/current/

Установить локацию:

POST /api/user-locations/set_location/

Body: {"location_id": 1, "is_auto_detected": false}

Автоматическое определение локации:

POST /api/user-locations/detect_location/

Body: {"latitude": 55.7558, "longitude": 37.6176}
